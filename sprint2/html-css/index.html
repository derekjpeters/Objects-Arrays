Introduction to CSS Styles

What is CSS?

CSS (Cascading Style Sheets) is a language used to describe the presentation of a document written in HTML or XML.

The Role of CSS in Web Development

CSS enhances the look and feel of web pages and ensures a consistent style across multiple pages.

Syntax of CSS

A CSS rule consists of a selector, property, and value:

p {
  color: blue;
  font-size: 14px;
}

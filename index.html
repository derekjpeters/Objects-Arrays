JavaScript Objects Challenges

1. The Pet Store

Task: Create objects to represent different pets in a pet store.

  • Properties:
    • type (dog, cat, bird, etc.)
    • name
    • age
    • breed (optional)
    • color
  • Method: Add a makeSound() method.

2. Student Records

Task: Create objects to represent different students in a school.

  • Properties:
    • name
    • age
    • gradeLevel
    • courses
  • Method: Add a study() method.

3. Recipe Manager

Task: Create objects to represent different recipes in a cookbook.

  • Properties:
    • name
    • ingredients
    • instructions
  • Method: Add a cook() method.

4. Library System (Challenge Mode)

Task: Create objects to represent different books in a library system.

  • Properties:
    • title
    • author
    • yearPublished
    • genre
  • Method: Add a checkout() method.

5. Vehicle Factory

Task: Create objects to represent different vehicles in a factory.

  • Properties:
    • type (car, truck, motorcycle, etc.)
    • model
    • year
    • color
  • Method: Add a startEngine() method.

6. Book Collection

Task: Create objects to represent different books in a collection.

  • Properties:
    • title
    • author
    • yearPublished
    • genre
  • Method: Add a read() method.

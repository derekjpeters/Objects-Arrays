The flex container

Imagine you're a web developer for a high-traffic e-commerce site. With Black Friday approaching, your team decides to revamp the product listings to make them more appealing and accessible on various devices. Utilizing Flexbox, you can quickly create a responsive and organized layout to showcase products effectively, ensuring a seamless shopping experience.

Code:

    div.parent {
      display: flex; /* Defines the element as a flex container */
      justify-content: center; /* Centers items horizontally within the container */
      align-items: center; /* Centers items vertically within the container */
    }

Distributing flex items along the main axis

As a front-end developer at a tech startup, you're told to design a user dashboard where notifications, messages, and daily tasks must be evenly spaced for an intuitive user interface. With Flexbox's 'justify-content' property, you efficiently implement this design, optimizing user experience and workflow.

Code:

    div.container {
      display: flex; /* Defines the element as a flex container */
      justify-content: space-around; /* Distributes items evenly with space around them */
    }

Flex items

While working on a media website, you must format article previews so images align to the top and the text to the bottom within their containers. By understanding and manipulating flex item properties, you set each item's alignment individually to achieve the desired layout.

Code:

    div.item {
      align-self: flex-end; /* Overrides the align-items property to align this item to the end of the container's cross axis */
    }

Distributing flex items along the cross axis

Your client has tasked you with creating a promotional web banner with text and images that need to be centered within a horizontal layout. With the understanding of the cross axis in Flexbox, you can quickly center items vertically, adjusting even when additional text or images are added.

Code:

    div.container {
      display: flex; /* Defines the element as a flex container */
      align-items: center; /* Centers items vertically within the container */
    }

The flex container (footer example)

You are designing a footer for a website where various links must be spaced out evenly. Your challenge is to create a layout that works across different screen sizes and aligns elements neatly, which can be typically challenging. By utilizing Flexbox properties on the container, you create a responsive footer that maintains its structure and spacing no matter the screen size.

Code:

    div.footer-container {
      display: flex; /* Defines the element as a flex container */
      flex-direction: row; /* Aligns items in a row (default flex direction) */
    }
